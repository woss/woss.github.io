## Key Projects

### Anagolay Network

A blockchain-based rights management protocol I founded as CTO and Protocol Evangelist. Designed in TypeScript and Rust, built on Substrate. Secured grants from Web3 Foundation, Parity Technologies, and the EU Next Generation Internet initiative. Proved I can learn anything from zero — went from no blockchain knowledge to protocol designer and grant-winning founder. [github.com/anagolay](https://github.com/anagolay)

### Kelp.digital / Macula

A content rights and multimedia management platform — the purest expression of who I am as a builder. Kelp uses cryptography to issue legally binding digital certificates for intellectual property. Macula is the centralized content rights management platform evolved from Anagolay's blockchain work. Built with SvelteKit, Sharp, ExifTool, IPFS, QuestDB, and SurrealDB. Serves photographers and creators with tools to verify, store, and license their work. [kelp.digital](https://kelp.digital) · [macula.link](https://macula.link)

### DaliORM & DaliMemory

A type-safe ORM for SurrealDB with full query builder, migration system, and schema inference — all pure TypeScript, no code generation. DaliMemory extends it as an AI memory tool for SurrealDB. Published as `@woss/dali-orm` and `@woss/dali-memory` on npm. [github.com/woss/dali](https://github.com/woss/dali)

### woss.io

An AI-native personal portfolio and chat platform built with SvelteKit 2, Svelte 5, and Tailwind CSS v4. Features streaming LLM chat, local vector embeddings with Transformers.js, RAG pipeline, and MCP tool integration for GitHub and Macula. [woss.io](https://woss.io) · [github.com/woss/woss.io](https://github.com/woss/woss.io)

### aws-lambda-layer-sharp

AWS Lambda Layer providing the Sharp image processing library without building or shipping the module. Just include and use. 20 stars on GitHub. [github.com/woss/aws-lambda-layer-sharp](https://github.com/woss/aws-lambda-layer-sharp)

### git-gpg-remote-sign

Tool for signing git commits and tags with GPG keys stored on remote servers — including Substrate keys. 5 stars. [github.com/woss/git-gpg-remote-sign](https://github.com/woss/git-gpg-remote-sign)

### exiftool-action

GitHub Action wrapping ExifTool for metadata extraction in CI/CD pipelines. Supports multiple ExifTool versions. 2 stars. [github.com/woss/exiftool-action](https://github.com/woss/exiftool-action)

### elmo

Efficient Link Management and Organization tool — built entirely on IPFS for decentralized storage and retrieval of links and content. [github.com/woss/elmo](https://github.com/woss/elmo)

### opencode-visualizer

Dashboard for visualizing OpenCode sessions, token usage, and AI coding activity. [github.com/woss/opencode-visualizer](https://github.com/woss/opencode-visualizer)

### graphql-to-sql

GraphQL model to SQL transpiler for Postgres — originally built as a Hasura integration tool. 13 stars. [github.com/woss/graphql-to-sql](https://github.com/woss/graphql-to-sql)
